# grpc-test-sample

The gRPC server test sample repository in Go.
</br>
</br>
Check [this](https://medium.com/@3n0ugh/how-to-test-grpc-servers-in-go-ba90fe365a18) blog to learn how to write test for gRPC servers.

# Instructions

- To run tests:
```bash
  go test ./... -v
```

- To run server:
```bash
  go run ./server
```

- To run client:
```bash
  go run ./client
```
